# Calculator

## 📌 Project Description
This project is a simple and responsive calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an easy-to-use interface.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear (AC) button
- Delete (DEL) button
- Responsive user interface
- Real-time calculation display

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure


## ▶️ How to Run
1. Download or clone this repository.
2. Open the `OIBSIP-L2-Task1-Calculator` folder.
3. Open `index.html` in any modern web browser.
